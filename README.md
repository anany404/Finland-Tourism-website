# Finland Tourism Website

This is a static travel and tourism website focused on Finland, designed to showcase places to visit, things to do, and cultural highlights of the country.

The site is deployed on GitHub Pages and accessible at:  
https://anany404.github.io/Finland-Tourism-website/

---

## Overview

The website provides information and inspiration for people planning a trip to Finland. It features:

- Destination highlights
- Things to do and places to explore
- Travel tips and practical information
- Cultural insights and seasonal travel ideas

Content is laid out in a mobile-friendly format with clear navigation.

---

## Technologies Used

This site is built using:

- HTML for structure
- CSS for styling
- JavaScript for interactivity
- Bootstrap for responsive design
- Webfonts for typography

All assets are stored locally in the project.

## How to Use the Repository

1. Clone or download the repository to your local machine.
2. Open the main project folder.
3. Open the `index.html` file in your browser to view the site locally.

No server or installation is required.
